Vocabulary

"Findability Vocabulary

SEO
Search Engine Optimization: the process of optimizing a website to increase its visibility in search engine results.
Meta Tags
HTML elements that provide metadata about a webpage, used by search engines to understand the content of the page.
Indexing
The process by which search engines collect, parse, and store data to include in their search results.
Keywords
Specific words or phrases that describe the content of a webpage and are targeted for search engine optimization.
Backlinks
Links from other websites to a particular webpage, considered as a crucial factor in determining a site's ranking in search engines.
Canonical URL
The preferred version of a URL that search engines should index when multiple URLs contain the same or very similar content.
Robots.txt
A file on a website that instructs search engine crawlers which pages or files they can or cannot request from the site.
Sitemap
A file that lists all the URLs of a website, used to help search engines crawl and index the site more effectively.
Alt Text
Descriptive text used within an HTML image tag that describes the content and purpose of an image for accessibility and SEO purposes.
Click-through Rate (CTR)
The ratio of users who click on a specific link to the number of total users who view a page, often used to measure the success of an online advertising campaign.
